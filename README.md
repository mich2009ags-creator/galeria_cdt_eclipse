# galeria_cdt_eclipse
Projeto final galeria
🎨 Eclipse Gallery
Uma galeria de arte digital moderna criada para expor fotografias de forma elegante e imersiva, unindo estética visual, tons de azul e uma experiência sofisticada para o usuário.
"Onde a luz e a câmera se encontram."
✨ Sobre o projeto
O Eclipse Gallery foi desenvolvido com o objetivo de apresentar fotografias artísticas em um ambiente moderno, responsivo e visualmente marcante.
O projeto busca transmitir uma identidade estética inspirada em museus e galerias contemporâneas, destacando as imagens de forma limpa e profissional.
🖼️ Funcionalidades
Galeria de imagens interativa
Design moderno e minimalista
Interface responsiva
Organização visual elegante
Navegação simples e intuitiva
Estética inspirada em exposições artísticas
